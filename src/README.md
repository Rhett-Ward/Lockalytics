# This is an explanation of the zip file

it is a zip file because the file structure matters.

This container was built using this guide: https://towardsdev.com/setting-up-an-hdfs-docker-cluster-a-step-by-step-guide-d4846ff54b5d

The zip file additionally contains the following:
- Everything you need thats not in the requirements

- My HTML file used to show functionality
- The Node.JS file that makes the html using data from the MongoDB
- The requirements.txt that is installed in the pyenv in order to run the python api script
- The package.jsons for the Node.JS file
- $DATANODE_DIR which was created when I first ran the container.
